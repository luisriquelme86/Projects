- Objetivos
. Crear un documento HTML que conste de un título h1 con el texto <<Pulsa una flecha>>, y un div de 300 por 300 px con borde color negro, 
  color de fondo amarillo y márgenes de 200px en todas direcciones. Utilizar el evento keypress para determinar si el usuario pulsa una tecla del cursor, y en ese caso utilizando CSS 
  haz que el cuadrado que define el div se desplace 20px en la dirección de flecha elegida por el usuario. Por ejemplo, si el usuario pulsa la flecha derecha, 
  el div debe desplazarse 20 px dentro de la ventana, hacia la derecha
